# KShader-Project

个人用于收录一些学习Shader过程中的项目合集

## 项目描述

- 编辑器: Unity 2022.3.14f1c1
- 主要语言：C#
- 代码编辑器：JetBrains Rider 2023

## 提交规范
- feat ：新功能（feature）
- fix ：修补Bug
- docs：文档
- style：格式（不影响代码运行的变动）
- refactor：重构
- test：增加测试
- chore：构建过程或辅助工具的改动

> 若为 **feat** 和 **fix** ，则该 **commit** 将肯定出现在 **Change Log** 之中
